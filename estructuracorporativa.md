# Estructura Comercial y Operativa

## 1. Desarrollo de Software a la Medida (Sistemas Empresariales)
[cite_start]**Concepto:** Creación de herramientas complejas (Inventarios, CRMS, ERPs, Puntos de Venta) adaptadas a las reglas de negocio del cliente[cite: 2, 4].

[cite_start]Ofrecemos al cliente dos modalidades de contratación para el mismo software[cite: 5]:

### Modalidad A: Venta de Código (Inversión Única)
[cite_start]*Ideal para empresas grandes que quieren tener control total y activo fijo.* [cno es´tas siite: 6, 7]
* [cite_start]**Modelo:** El cliente paga el desarrollo completo[cite: 11].
* **Propiedad:** El cliente es dueño del código fuente. [cite_start]Se le entrega todo al finalizar[cite: 12].
* **Costo Inicial:** Alto (Cotización completa). [cite_start]Ejemplo: Si el sistema vale Q. 25,000, pagan eso (generalmente 50% anticipo, 50% entrega)[cite: 12, 13].
* **Costos Recurrentes (Opcional):** El cliente no está obligado a pagarnos mensualmente. [cite_start]Si desean soporte o actualizaciones futuras, deben contratar un Plan de Mantenimiento o pagar por hora[cite: 14, 15, 16].

### Modalidad B: Software como Servicio - SaaS (Inversión Inicial + Renta)
[cite_start]*Ideal para PYMES que no quieren descapitalizarse o lidiar con servidores.* [cite: 17, 18]
* [cite_start]**Modelo:** El cliente paga una implementación y una "renta" por uso[cite: 19].
* **Propiedad:** La Agencia mantiene la propiedad del código. [cite_start]El cliente solo tiene una licencia de uso mientras pague[cite: 20].
* **Costo Inicial (Setup Fee):** Moderado / Variable. Cubre instalación, configuración servidor, personalización logos y capacitación. (Rango estimado: Q. 3,000 a Q. 8,000) [cite_start][cite: 21, 22, 23].
* **Costo Recurrente (Suscripción):** Mensualidad Obligatoria. Cubre hosting, licencia, backups y soporte técnico directo. (Rango estimado: Q. 400 a Q. 1,200 al mes) [cite_start][cite: 23, 24, 25].

## 4. Políticas y Condiciones de Trabajo
1.  **Forma de Pago:**
    * [cite_start]Proyectos: 50% anticipo, 50% contra entrega[cite: 59].
    * [cite_start]Suscripciones: Pago anticipado mes a mes[cite: 60].
2.  **Propiedad Intelectual:**
    * [cite_start]Modalidad A (Venta): Al liquidar, cliente recibe código fuente y credenciales[cite: 62].
    * Modalidad B (SaaS): Código es propiedad de la Agencia. [cite_start]Cliente paga por acceso[cite: 63, 64].
3.  **Alcance:** Definido por la cotización. [cite_start]"Nuevas ideas" son extras[cite: 65, 66].
4.  [cite_start]**Suspensión:** Tras 15 días de mora en pagos mensuales, el servicio se suspende[cite: 68].
5.  **Tiempos de Respuesta:**
    * [cite_start]Lunes a Viernes: Requerimientos y dudas[cite: 70].
    * [cite_start]Fines de Semana: Exclusivamente emergencias críticas (caída total)[cite: 72].
